# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Instead, please email: **security@yourdomain.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if you have one)

We'll respond within 48 hours and work with you to address the issue.

## Security Best Practices

When using this calculator:
- Never share your Polymarket wallet private keys
- Don't store API credentials in code
- Use environment variables for sensitive data
- Be cautious with custom scripts that handle real money

## Disclaimer

This tool is for **educational purposes only**. 

- Not financial advice
- Use at your own risk
- Always verify calculations independently
- Only invest what you can afford to lose
